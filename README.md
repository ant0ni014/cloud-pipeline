# Continuous Cloud Deployment Pipeline

Ein durchgängiges DevOps- & Cloud-Projekt zur automatisierten Bereitstellung einer Webanwendung auf der T Cloud Public.

## Tech Stack
* **Language:** Python
* **Containerization:** Docker
* **CI/CD:** GitHub Actions
* **Cloud Platform:** T Cloud Public (ECS / Ubuntu)

## Features
* Leichtgewichtige Python HTTP-Webanwendung
* Docker-Container für konsistente Laufzeitumgebung
* Automatisierte Build- und Test-Pipeline via GitHub Actions
